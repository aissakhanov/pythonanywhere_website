Linear Regression

This page consists of 2 main parts: Demo(with 2 tables below) and explanatory notes. Let's go through each of them:

  • Demo: A graph, where y-axis is our target and x-axis is a feature which possibly explains our target values. A graph contains initial data(blue), regression line(orange), average point of feature and target values(red) and y-intercept(violet). YOU CAN add points by clicking on the graph, which results in green dots(your data). Consequently, regression line, y-intercept and average will automatically adjust for new data. In addition, there are control buttons on top right corner, which will help you to reset axes and download a graph as an image.
  • Tables below the graph: 1st table contains regression line parameters: y-intercept, slope, and R-squared. 2nd table contains initial and your data points. Both of these tables dynamically change as you add more data.
  • Explanatory notes: It contains main highlights/insights for this demo, additionally some Python code and math formulas written in LaTex. Hence, this section is done in Google Colab notebook, for convenience.
